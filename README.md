# Модульные инфобоксы FANDOM
Различная информация по модульным инфобоксам FANDOM.

### Создание темы

```xml
<!-- Страница шаблона -->

<infobox type="название_темы">
	...
</infobox>
```

```css
/* MediaWiki:Wikia.css или MediaWiki:Themes.css */

.portable-infobox.type-название_темы {
	...
}
```
